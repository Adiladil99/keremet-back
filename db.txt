Если вы установили MySQL на macOS с использованием пакета (pkg), то для запуска MySQL-сервера вам следует выполнить следующие шаги:

1. **Открыть Терминал:**
   - Вы можете найти его через Spotlight (Cmd + Space и начните вводить "Терминал").

2. **Перейти в директорию с MySQL:**
   - По умолчанию MySQL устанавливается в `/usr/local/mysql/`. Введите следующую команду в Терминале:

     ```bash
     cd /usr/local/mysql/bin
     ```

3. **Запустить MySQL-сервер:**
   - Для запуска MySQL-сервера используйте следующую команду:

     ```bash
     sudo ./mysqld
     ```

   - Вам может потребоваться ввести пароль администратора macOS.

4. **Открыть новое окно Терминала (опционально):**
   - В новом окне Терминала перейдите в директорию MySQL bin:

     ```bash
     cd /usr/local/mysql/bin
     ```

5. **Запустить клиент MySQL:**
   - Теперь вы можете запустить клиент MySQL с помощью следующей команды:

     ```
     bash
     ./mysql -u root -p
     ```

     - Если вы впервые запускаете MySQL, вы можете использовать `-u root` для входа под учетной записью администратора (root).
     - После этого система может запросить вас ввести пароль для пользователя root.

6. **Завершение работы:**
   - Чтобы остановить MySQL-сервер, вернитесь в окно Терминала, где вы запустили `mysqld`, и используйте команду `Ctrl+C` для завершения работы сервера.

Теперь у вас должен быть запущен MySQL-сервер на вашем Mac. Пожалуйста, обратите внимание, что эти инструкции могут потребовать некоторых адаптаций в зависимости от конкретной версии MySQL и конфигурации вашей системы.
